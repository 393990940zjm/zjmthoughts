你好呀~<p>
<br/>
不知道当你看到这封信的时候，是我们还在逛街呢，还是你已经快要睡了<p>
这是我送你的一份礼物，有些不太好表达的话，或者一些生活的经历和奇奇怪怪的想法，我都会写下来。然后放在上面和你一起分享。不过我不知道你对这样的方式反不反感耶，比如你已经蛮久没有好好看过书了。<p>
<br/>
算起来，今天是我们认识的第5天了呢，缘分真的好奇妙呀。<p>
<br/>
今天要和你见面，有点小紧张耶。不知道我会不会是你喜欢的类型，别一见面就见光死哈哈哈。不过呢，我虽然长得不帅，不过应该也还算看得过去啦。<p>
<br/>
我有想象和你见面以后，我们聊了哪些话题，走过哪些地方，看着你被我逗笑的样子，觉得很温馨，很幸福。<p>
<br/>
不知道为什么，和你聊的第一天开始，就有一种彼此很熟悉的感觉。好像我们就是最适合彼此的那个人，一样的爱好，一样的观点，一样的铁杆粉丝。仿佛这么多年，我们就是在等待相遇的这一天。<p>
<br/>
我是个很重感情的人，我会为感情的事情付出很多，思考很多。我会尽力会经营这段难得的感情，不让彼此失望，不让彼此伤心，每一天都是在一起快乐的一天。我希望自己的另一伴和我在一起是舒服的，是温暖的，是开心的，是放松的。我们可以一聊就聊一整夜，也可以什么都不说，但看着对方的时候，彼此心里都有感觉。<p>
<br/>
这是我给你写的第一封信，未来可能是一周，或者两到三天，我会继续把心里的话写下来。<p>
<br/>
未来我们没法预测，但如果可以，我想陪你一起走过未来的人生~<p>

毕竟是第一封信，就先写到这吧哈哈~<p>
