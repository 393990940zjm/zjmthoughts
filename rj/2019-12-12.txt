<p>
今天，聊聊我的过去吧，我觉得你应该会感兴趣。<p>
<br/>
小的时候，我是个很皮的人。上蹿下跳，为非作歹，丝毫没有埋没我属相的光辉。<p>
<br/>
很喜欢玩火，差点把家里的缝纫机烧了。动不动就和小伙伴去偷隔壁菜场的瓜啦，鸡蛋啦，然后跑到附近的野地里，点柴火烤着吃。当然了，旁边的地里有熟的蔬菜也逃不过我们的眼睛。有时候一不小心，就点着了一大片枯地。眼看着火势越来越大，扑灭无望，小伙伴们相互一视，便逃的无影无踪。<p>
<br/>
很喜欢做危险的事情，在铁轨上放石头、在矮房上爬来爬去、在楼顶上尿尿等等。经常是出去没疯多久，隔壁邻居就跑来砸门找我外公，“沈师傅，你外孙又钻到车子底下去了...”，“沈师傅，你外孙又把别人弄开了”，“沈师傅，你外孙又把我家窗户弄破嘞”，“沈师傅，你外孙......哦......今天你外孙没来......”<p>
<br/>
那个时候，学习也不好，生活作风也不好。老师什么事不让做，我们就偏偏要去做。逃课去游戏厅，偷小店的糖果，考试作业乱写，现在想来真是无敌了。这样的无法无天持续到了6年级才开始有了好转，可能是在逐渐长大的过程中，知道了面子这件事，知道羞耻了，所以慢慢收敛了起来。说起6年级，有一件事还挺值得思考的，等见面的时候和你说，记得提醒我。<p>
<br/>
上了初中之后，喜欢上了曾经最讨厌的创作，开始写小诗，写文章，班级还专门给我出了一个黑板报。那是我文学路上的巅峰，也是启蒙。后来，很多时间都花在写情书上，没有很用功的学习功课，结果考了470分，上了技校。<p>
<br/>
我学习的是数控技术，和现在的工作还算有那么一点缘分。但是技校嘛，毕竟是差等生的常驻地，关系复杂，偶尔还是会看到集体罢课或者打架斗殴的场面。那个时候，作为班级里的团支部书记，其实并没有起到太多作用。只是每个学期拿拿奖学金，考试的时候给邻桌抄一抄答案，对了，还顺便拿了一个全国作文一等奖和三等奖。其他也并没有什么可谈论的了。<p>
<br/>
2012年，我终于迈上了社会这个大家庭，开启了工作的新旅程。<p>
<br/>
不过太稚嫩，毫无经验，情商智商堪忧。所以看着别人都一步一步到了好的岗位，自己还留在原地，干着最底层的工作。很无奈，也很不甘心这样埋没。于是当我想好要做什么的时候，我终于离开了那里。那一年，是2016年。<p>
<br/>
经过了4个月的学习，我到了现在的公司。<p>
<br/>
一开始也挺辛苦的，但是还年轻，干活的劲头很足，所以经常很晚了还在工作。因为基础并不是很扎实，所以自己也花了很多时间学习，慢慢的开始接手了更多工作，也带了很多新人，工资也慢慢高起来了。就这样带着成就感一直干啊干啊，干到了今天。<p>
<br/>
内容有限，我讲的比较粗略，很多细节等我们以后有大把的时间去聊。<p>
<br/>
蓉，其实我想告诉你，我们见面的时候，如果我问出一些可能你觉得幼稚的话题，我希望你可以不要笑。因为这些问题，可能我的经历相关，也能是想对你多一些了解。<p>
<br/>
蓉，有很多的心里话，我想在你的耳边悄悄对你说。可能是和感情有关，也可能是生活有关，所有当你发现我开始认真看着你眼睛的时候，你也可以收起调皮的性子，安安静静听我倾诉。<p>
<br/>
我是一个懂得自我反省的人，也是个会接受批评的人。所以以后有什么想法，或者觉得我哪里做得不对，你都可以和我说。我最怕的是我明明做了很多不该犯的错误，却因为你的宽容和忍让，让我一错再错。<p>
<br/>
好了，今天也说的不少了，就先这些吧。<p>
