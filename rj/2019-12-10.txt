<p>
早上好呀~<p>
<br/>
虽然这份信是晚上写的，并且是在睡前写的，但是真诚一丢丢也不会少。<p>
<br/>
写这份信之前，我去认真回顾了一下我们昨天的对话，也认真回忆了一下我们见面那天的情景。我有点生气那天你对我的态度，当然也是我做得不够妥当啦，不过我还是能够感受到你的体谅和宽容。说心里话，能遇到如你一般善良的姑娘，是我莫大的福分。<p>
<br/>
见面之后，其实有很多话想对你说，但是我没有办法说出口。我怕有些本无心的话一旦带上了埋怨的语气或者责备的态度，性质就变了，我害怕错失你，也害怕让你不开心。所以我选择了安静了两天，让你自己好好冷静一下，正视这段感情。<p>
<br/>
这两天，没有发生什么大的事情，一样的早起，一样的晚睡，一样吃完了饭就打瞌睡。有时候想到了和你在一起的画面，却不自觉的笑了起来。同事好奇的看着我，说你在发什么神经，是不是在想哪个美女。我很调皮的回他，我说，我看上你了。他露出了嫌弃又诡异的笑容，你走开，老子性取向还是蛮正常的，不好你这口。哈哈哈哈<p>
<br/>
蓉，我喜欢看你的眼睛，你的眼睛仿佛有灵性，泛着水光。当我注视着你的时候，好像整个世界都安静了，只剩下我和你。<p>
<br/>
在我们交往的这段时间，你应该对我也有一些了解。虽然第一次见面，我们都显得比较拘谨，但还是有很多说不完的话题，不太容易冷场。不过随着相处机会的增多，彼此信任的提高，我们终究会走到一起的。毕竟，我们心里都已经有了对方。<p>
<br/>
我是个对感情挺执着的人，会为了喜欢的人默默付出很多。比如我会去留意你说的一些话，你的生日，你的喜好还有厌恶的东西。我会悄悄去准备一些礼物，有时候是睡前一首安眠曲，有时候是一份生日祝福，有时候是情人节的礼物。<p>
<br/>
我有过在冬日的雨夜里，怀里揣着热奶茶等着下班回家的女生。<p>
<br/>
我有过精心准备了制作一天的礼物，然后送给一个付出已久但无法被接纳的女生。<p>
<br/>
可能这些都是不属于我的缘分，付出了很多，虽然离开了，有不甘，但也没什么可遗憾的。也许是我用错了方式，也许是她们要求的更多，但我如果再次遇到一个喜欢的人，我还是会这么做，可能是用别的方式，可能会变得很冲动。但我满心的喜欢，都是真实的。<p>
<br/>
有人说，女生都喜欢成熟稳重的男生。我在想什么是成熟，我思考了很久，我所能想到的是。所谓的成熟，和年龄无关，和性格无关，是当你真正明白了生活的真相之后，还是能够依旧热爱着生活，这才是真正的成熟。云卷云舒，波澜不惊。<p>
<br/>
蓉，我不知道你介不介意我这样称呼你。你对我来说，很重要，很特别。在我的记忆里，我已经很久没有和女生聊得这么开心舒服过了，不过我也不知道你是不是和谁都能这么聊得来。<p>
<br/>
如果可以，我想牵着你的手，走过落满樱花的街。<p>
<br/>
如果可以，我想彼此依靠，看着太阳在远方慢慢升起。<p>
<br/>
也许你会觉得文字里的这个男生更真诚，但现实中，我更想抱紧你，听我说着所有的真心话。<p>
<br/>
也许你会觉得我笑嘻嘻的不太正经，或者看起来是个不可靠，不能够信任的人。不过随着时间的推移，你终会发现，我们是最适合彼此的那个答案。<p>
<br/>
这两天里我设想过很多我们的未来，虽然都是假象，但都很温暖。我告诫自己，一定多留意你的感受和生活的细节，我不希望因为我的粗心，让未来本该幸福的我们，在每个不留意的生活细节里逐渐失望，一点一点死心，我不想发生这样的事。所以，如果我们在一起，我会更加珍惜这份感情。<p>
<br/>
可能是有太多的话要说吧，一不小心，就写了这么好多。我觉得你可能已经看着累死了哈哈哈。<p>
<br/>
先到这里吧，毕竟来日方长，毕竟......太晚了呐，困死了哈哈哈，睡了睡了。<p>
<br/>
不对，你看到已经是早上了，又是元气满满的一天呢，加油干哈哈哈哈~<p>
